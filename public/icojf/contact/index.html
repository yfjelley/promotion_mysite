Project Fit

适合与不适合的项目

适合进入评估

  • 已有业务流程，需要系统化或自动化。
  • 已有产品方向，需要 SaaS MVP 验证。
  • 已有第三方平台或 API 文档，需要稳定集成。
  • 愿意按阶段交付，并接受验收清单。

不适合进入开发

  • 没有明确负责人或验收人。
  • 只要求复制竞品，没有业务流程和数据边界。
  • 不接受源码、部署文档、运行手册和验收清单交接。
  • 希望承诺融资、增长、广告 ROI 或第三方平台审核结果。
