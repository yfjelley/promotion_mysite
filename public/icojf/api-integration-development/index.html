上线前检查

API 项目最重要的不是“能调通一次”，而是失败时可追踪、可恢复、可解释。

权限

确认最小权限、密钥存放、IP 限制、密钥轮换和生产/测试隔离。

可靠性

确认重试、去重、超时、限频、分页、增量同步和补偿任务。

验收

用真实样例数据检查字段、状态流转、失败日志和异常通知。
