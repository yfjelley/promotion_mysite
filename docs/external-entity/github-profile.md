# SignalCraft Labs

SignalCraft Labs is a remote software development studio for automated trading
systems and trading API workflows.

Website: https://pddjf.com/  
Contact: contact@pddjf.com

## What We Build

- TradingView Webhook automation
- Exchange API order execution workflows
- Broker API automation for IBKR, Schwab, Alpaca, FIX, REST, and WebSocket APIs
- Risk checks, order routing, logs, alerts, and private deployment

## Service Boundary

SignalCraft Labs provides technical software development services only. We do
not provide investment advice, trading signals, managed accounts, custody,
withdrawal permissions, or profit promises.

Third-party platforms mentioned in this repository or on the website are
integration targets only. SignalCraft Labs does not claim official partnership,
authorization, or endorsement unless explicitly stated in writing.
